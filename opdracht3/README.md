# frontend voor designers

Opdracht 3

Voor opdracht 3 heb ik Activiteiten in Amsterdam API gekozen om data te krijgen.

Ik heb gekozen om een carrousel te maken die de verschillende activiteiten laat zien. Op deze manier kan de gebruiker makkelijk verschillende activiteiten vinden door te swipen.

Bovendien heb ik gebruik gemaakt van een night-day vision knop, zodat de gebruiker altijd de mogelijkheid heeft. Om te kiezen welke vision het beste bij hem/ haar en bij een situatie past.

De content in de pagina is genomen van de API die ik had gekozen.
Ik heb gekozen om title, informatie, adres en foto's van de activiteiten
te laten zien.

- De gebruiker ziet een carrousel van verschillende kaarten die de content laten zien. Op de voorkant van een kaart ziet de gebruiker de title, informatie, adres, en een Image knop.

- Wanneer de gebruiker op de adres klikt, verschijnt een iframe (Google Maps) van de locatie. Dat heb ik gekozen om te doen omdat op deze manier de gebruiker kan de locatie toevoegen aan zijn/ haar google maps. En een route maken van verschillende activiteiten.

- Als de gebruiker op de IMG icon hovert (mouseover), flipt de kaart naar de achterkant en krijgt de gebruiker een foto van een activiteit te zien.


Flip-card animatie bron: https://www.w3schools.com/howto/howto_css_flip_card.asp
